# Agents Board

Het Agents Board toont alle tickets per agent.

## Kolommen
Elke agent heeft een eigen kolom. 
Er is ook een kolom "Niet toegewezen" 
voor tickets zonder agent.

## Ticket toewijzen
Sleep een ticket naar de kolom van een agent. 
De status verandert automatisch naar 
"In behandeling".

## Alleen open tickets
Gesloten tickets verschijnen niet op 
het Agents Board.
