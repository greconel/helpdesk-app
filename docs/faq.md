# Veelgestelde vragen

## Hoe maak ik een ticket aan?
Ga naar /tickets/create, vul het formulier 
in en klik op versturen.

## Hoe weet ik aan wie een ticket toegewezen is?
Op de ticketpagina zie je rechts onder 
"Toegewezen aan" wie verantwoordelijk is.

## Kan ik een ticket opnieuw openen?
Ja. Wanneer een klant reageert via e-mail 
op een gesloten ticket wordt de status 
automatisch teruggezet naar "Nieuw".

## Wat betekent het AI icoontje?
Het paarse "AI" icoontje betekent dat de 
impact of labels automatisch ingesteld 
werden door de AI.

## Hoe log ik tijd op een ticket?
Open het ticket en ga naar "Tijd loggen" 
in de rechterzijbalk. Start een timer 
of voer manueel uren en minuten in.

## Wat is het verschil tussen 
## Status Board en Agents Board?
Het Status Board groepeert tickets per status. 
Het Agents Board groepeert tickets per agent.
