# Tickets

Een ticket is een melding of vraag van een klant 
die opgevolgd moet worden door een agent.

## Ticket aanmaken

### Als klant
Ga naar /tickets/create. Vul je naam, e-mailadres, 
onderwerp en beschrijving in en klik op versturen.
Je ontvangt een bevestigingsmail met je ticketnummer.

### Als agent
Ga via het menu naar "Ticket aanmaken". Als agent 
kan je meer informatie invullen zoals impact, labels 
en aan welke agent het ticket toegewezen wordt.

## Ticketnummer
Elk ticket krijgt automatisch een uniek nummer 
zoals #0001, #0002, enzovoort.

## Statussen
- **Nieuw**: ticket is net aangemaakt
- **In behandeling**: een agent is actief bezig
- **On hold**: gepauzeerd, wachten op meer info
- **Te sluiten**: opgelost maar nog niet gesloten
- **Gesloten**: volledig afgehandeld

## Ticket toewijzen
Via het Agents Board sleep je een ticket naar 
de juiste agent. De status verandert automatisch 
naar "In behandeling".
