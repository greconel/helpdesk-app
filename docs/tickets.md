# Tickets

Een ticket is een melding of vraag van een klant 
die opgevolgd wordt door een agent in het helpdesk systeem.

## Ticket aanmaken als klant

Er zijn drie manieren om een ticket aan te maken als klant:

### Via het webformulier
Ga naar /tickets/create. Vul de volgende velden in:
- Naam
- E-mailadres
- Onderwerp
- Beschrijving

Na het versturen ontvang je automatisch een 
bevestigingsmail met je ticketnummer.

### Via e-mail
Stuur een e-mail naar de helpdesk. 
Het systeem leest automatisch elke minuut 
de inkomende e-mails uit en maakt daar 
een ticket van aan. De inhoud van je 
e-mail wordt de beschrijving van het ticket.

### Antwoorden op een bestaand ticket
Wanneer je antwoordt op een bevestigingsmail 
of een bericht van een agent, wordt je 
antwoord automatisch toegevoegd aan het 
bestaande ticket. Het ticket wordt 
heropend als het al gesloten was.

## Ticketnummer

Elk ticket krijgt automatisch een uniek nummer 
zoals #0001, #0002, enzovoort. Dit nummer staat 
ook in de bevestigingsmail en in het onderwerp 
van alle e-mails over dat ticket.

## Ticket aanmaken als agent

Agents kunnen tickets aanmaken via het menu 
"Ticket aanmaken". Als agent heb je meer opties:

- Bestaande klant selecteren via zoekfunctie
- Nieuwe klant aanmaken met naam, e-mail en telefoon
- Impact instellen (low, medium, high)
- Labels toewijzen
- Ticket direct toewijzen aan een agent
- Kiezen of er een bevestigingsmail verstuurd wordt

Wanneer een ticket direct toegewezen wordt aan 
een agent, wordt de status automatisch ingesteld 
op "In behandeling".

## Ticketnummer

Elk ticket krijgt automatisch een uniek nummer 
zoals #0001, #0002, enzovoort. Dit nummer staat 
ook in de bevestigingsmail.

## Statussen

Een ticket doorloopt de volgende statussen:

- **Nieuw**: ticket is net aangemaakt en nog 
  niet opgepikt door een agent
- **In behandeling**: een agent is actief 
  bezig met het ticket
- **On hold**: ticket is gepauzeerd, 
  bijvoorbeeld wachten op meer informatie 
  van de klant
- **Te sluiten**: het probleem is opgelost 
  maar het ticket is nog niet formeel gesloten
- **Gesloten**: ticket is volledig afgehandeld

## Ticket toewijzen aan een agent

Tickets toewijzen gebeurt via het Agents Board 
door een ticket te slepen naar de kolom van 
de gewenste agent. Wanneer een ticket toegewezen 
wordt aan een agent verandert de status 
automatisch naar "In behandeling".

## Ticket beantwoorden

Agents kunnen rechtstreeks vanuit het ticket 
een bericht sturen naar de klant via de 
communicatietijdlijn. De klant ontvangt 
dit bericht als e-mail. Wanneer de klant 
antwoordt via e-mail wordt dat antwoord 
automatisch toegevoegd aan de tijdlijn.

## Ticket heropenen

Wanneer een klant reageert via e-mail op 
een gesloten ticket wordt de status 
automatisch teruggezet naar "Nieuw".

## Tijd loggen op een ticket

Agents kunnen tijd loggen op een ticket 
via de rechterzijbalk op de ticketpagina. 
Er zijn twee manieren:

- **Timer**: start een timer en stop die 
  wanneer je klaar bent. De tijd wordt 
  automatisch berekend.
- **Manueel**: vul zelf uren en minuten in.

Alle gelogde tijd is zichtbaar in de 
rechterzijbalk met een totaal onderaan.

## Wijzigingen bijhouden

Elke wijziging aan een ticket wordt bijgehouden 
in de tijdlijn rechtsonder op de ticketpagina. 
Je ziet wie wat wanneer aangepast heeft, 
zoals statuswijzigingen, toewijzingen 
en impactwijzigingen.
